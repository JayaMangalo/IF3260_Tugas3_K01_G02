Help Menu

A. Left Sidebar

1. Help Button

Show this help menu.

2. Save Button

Save current object to local storage.

3. Load Button

Load saved object from local storage.

4. Camera Settings

Change camera settings.

  • Camera Radius: Change camera distance from object.
  • Camera Angle: Change camera angle from object.
  • Projection Mode: Change camera projection mode.
  • Reset Camera Settings: Reset camera settings to default.

5. Animate

Animate object to rotate in y-axis.

6. Shading

Turn on/off shading object.

B. Right Sidebar

1. Rotate

Rotate object in x, y, or z axis.

  • X: Rotate object in x-axis.
  • Y: Rotate object in y-axis.
  • Z: Rotate object in z-axis.

2. Translate

Translate object in x, y, or z axis.

  • X: Translate object in x-axis.
  • Y: Translate object in y-axis.
  • Z: Translate object in z-axis.

3. Scale

Scale object in x, y, or z axis.

  • X: Scale object in x-axis.
  • Y: Scale object in y-axis.
  • Z: Scale object in z-axis.
